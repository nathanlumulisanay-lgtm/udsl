# UDSL Examples (v1.0)

This directory contains sample documents generated under the UDSL v1.0 standard.
> **Disclaimer:** This is a fictional example document used solely to
> demonstrate UDSL v1.0 structure and formatting. It does not represent real
> policy, recommendations, analysis or advice.


## Included
- **policy_memo** → example of a compliant memo
- **technical_report** → structured analysis format
- **business_report** → executive business document
- **comparison** → cross-model outputs (ChatGPT, Claude, Gemini, Perplexity)

## Purpose
Examples accelerate adoption by showing:
- how UDSL structures long-form documents
- how reasoning schemas are integrated
- how consistent tone and terminology are applied
- how different AI models behave under the same rules

These examples are free to use, remix and reference under **CC BY 4.0**.
